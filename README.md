# 🎮 Childhood Dreams Donation Site 🚗

A nostalgic 90s-styled donation platform where people can fund childhood experiences they never had - both for current kids and adults who missed out.

## 🌟 Features
- 90s nostalgic design with authentic retro styling
- Razorpay payment integration
- AI-powered personalized thank you emails (Gemini Pro)
- Dual donation categories (Kids & Adults)
- Emotional storytelling and connection

## 🚀 Tech Stack
- Next.js 13
- Razorpay Payment Gateway  
- Google Gemini Pro API
- Nodemailer for emails
- 90s CSS styling with animations

## 💫 Mission
"Healing childhood disappointments by creating joy for others"
